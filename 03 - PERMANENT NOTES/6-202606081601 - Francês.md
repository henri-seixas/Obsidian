## Stack do projeto

| Componente                        | Ferramenta                            |
| --------------------------------- | ------------------------------------- |
| Flashcards com repetição espaçada | **Anki** (import `.txt` tab-separado) |
| Áudio de pronúncia                | **Piper TTS** (servidor home lab)     |
| Sincronização de arquivos         | **Nextcloud** via **Tailscale**       |
| Documentação e reflexão           | **Obsidian** (método Zettelkasten)    |
| Modelo de voz                     | `fr_FR-upmc-medium`                   |

### Como o método funciona

```
Arquivo .txt com frases
        ↓
  Anki (frente: frase com palavra em NEGRITO)
  (verso: tradução + forma da palavra)
        ↓
  Piper TTS gera áudio de cada frase
        ↓
  Revisão diária no Anki + escuta do áudio
        ↓
  Registro de progresso nesta nota
```

## 🧠 Princípios Zettelkasten Aplicados ao Estudo de Idiomas

### 1. Atomicidade
Cada palavra é uma unidade de conhecimento. A frase é o contexto mínimo necessário. O cartão Anki é a ancoragem dessa unidade.

> Não tente aprender "francês". Aprenda *cette phrase*, depois *cette autre phrase*.

### 2. Conexão
Ao estudar uma palavra nova, perguntar-se:
> "Onde já vi algo parecido? Em que situação usaria isso no mundo real?"

Registre conexões usando `[[links]]` nas notas de cada dia.

### 3. Emergência
Após 10 dias consistentes, padrões gramaticais emergem **sem que você precise estudá-los explicitamente**. O cérebro infere regras a partir de exemplos. Confie no processo.

### 4. Revisão como reativação
Revisão não é repetição mecânica. É **reativação de redes neurais**. Cada vez que você recupera uma memória, ela fica mais forte e mais acessível.

### 5. Captura imediata
Quando uma palavra surgir no mundo real (música, filme, conversa), **anotar imediatamente** na nota do dia e no caderno pessoal
